## 6. 知情同意说明

### 研究目的

本次访谈用于了解自由职业者和小型工作室在项目范围、交付验收、需求变更和回款方面的真实流程和问题。

### 参与方式

访谈约 40～50 分钟。你可以拒绝回答任何问题，也可以随时停止。

### 录音

只有在你明确同意后才录音。录音用于研究整理，不会公开发布。

### 隐私

研究材料会尽量去除：

- 姓名；
- 客户名称；
- 公司机密；
- 项目敏感数据；
- 可直接识别个人的信息。

未经许可，不会公开原始录音、完整聊天记录或客户文件。

### 材料使用

若你展示报价、合同、交付或催款材料，我们只记录与流程有关的信息。任何保存或进一步使用需单独征得同意。

### 联系

如希望撤回材料或停止后续联系，可通过 [联系方式] 提出。
