# MilestoneFlow 用户调研工具包 V1.0

本工具包用于在完整 MVP 开发前验证：

- 项目范围失控是否高频；
- 阶段交付与客户验收是否存在真实问题；
- 正式需求变更是否值得使用；
- 验收与尾款是否存在强关联；
- 用户是否愿意让客户通过免注册链接确认；
- 用户是否愿意真实试用和付费。

包含：

1. 用户调研计划；
2. 深度访谈执行手册；
3. 用户调查问卷；
4. 调研记录与分析模板；
5. 用户招募与知情同意模板。

重要原则：

- 不得伪造访谈结果；
- 不得将“听起来不错”当作付费意愿；
- 优先询问过去真实行为；
- 至少完成 20 次深度访谈；
- 至少获得 2 个真实付款信号后，再进入完整 MVP 开发。
