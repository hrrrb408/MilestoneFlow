# MilestoneFlow 低保真可点击原型 V1.0

## 使用方式

1. 解压压缩包；
2. 双击 `index.html`；
3. 使用 Chrome、Edge、Safari 或 Firefox 打开。

无需安装依赖，无需启动服务器。

## 覆盖流程

- 工作台与关键商业事项；
- 项目当前商业基线；
- 报价编辑和客户确认；
- 交付版本和客户验收；
- 需求变更和客户确认；
- 应收看板、付款记录和逾期提醒。

## 推荐演示路线

```text
工作台
→ 新建报价
→ 客户确认报价
→ 项目当前商业基线
→ 提交设计稿 V2
→ 客户请求修改或通过
→ 创建会员积分变更单
→ 客户确认变更
→ 应收看板
→ 发送逾期提醒或记录付款
```

## 限制

- 数据为演示数据；
- 刷新页面后恢复默认状态；
- 不发送真实邮件；
- 不上传真实文件；
- 不调用真实 AI；
- 不处理真实资金；
- 不代表最终 UI 视觉。
